# Corkboard - A Simple Note-Taking App for Loose Thoughts of All Sorts

## Description

Corkboard allows a user to create notebooks and fill them with various notes. Journals, to-do lists, and brainstorming scrapbooks of all kinds are possible. Notes can be shared between notebooks, giving the user ultimate flexibility in organizing their thoughts.

## Usage

To use this app, just clone, run `rake db:migrate` and then run `shotgun`. Everything should be set up.

After creating an account, you can start creating any kind of notebook you like and filling it with any notes that you feel like!
